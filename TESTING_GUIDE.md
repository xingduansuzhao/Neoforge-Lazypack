# 制箭台箭矢生成功能测试指南

## 功能说明
本模组实现了当玩家站在制箭台上时自动生成箭矢的功能：
- 只有当玩家站在制箭台上时才会生成箭矢
- 每秒生成10个箭矢（每100毫秒生成1个）
- 箭矢会在制箭台上方生成并带有轻微的抛物线运动

## 测试步骤

### 1. 启动游戏
运行以下命令启动游戏客户端：
```bash
.\gradlew runClient
```

### 2. 进入世界
- 创建新的创造模式世界或加载现有世界
- 确保能够正常进入游戏

### 3. 放置制箭台
- 从创造模式物品栏中找到"制箭台"(Fletching Table)
- 在地面放置制箭台

### 4. 测试功能
- 站在制箭台上
- 观察是否开始生成箭矢
- 箭矢应该在制箭台上方生成并向四周散落
- 离开制箭台后箭矢生成应该停止

### 5. 验证日志
检查控制台输出，应该能看到类似以下的日志信息：
```
[AIMod/]: 在制箭台生成箭矢，玩家: Dev
```

## 功能特点

### 生成机制
- **触发条件**：玩家必须站在制箭台的正上方
- **生成频率**：每秒10个箭矢（精确控制）
- **生成位置**：以玩家当前位置为中心，紧贴玩家生成
- **跟随效果**：玩家在制箭台上移动时，箭矢会跟随玩家位置生成
- **物理效果**：箭矢具有轻微的向上运动和旋转效果

### 性能优化
- 使用时间戳控制生成间隔
- 只对站在制箭台上的玩家进行计算
- 定期清理离线玩家数据

### 用户体验
- 箭矢生成位置紧贴玩家，提高拾取效率
- 生成有适当的拾取延迟（5tick）
- 玩家移动时箭矢生成位置实时跟随
- 离开制箭台立即停止生成

## 故障排除

如果功能不工作，请检查：
1. 控制台是否有错误信息
2. 是否确实站在制箭台正上方（不是旁边）
3. 模组是否正确加载（查看启动日志中的AIMod信息）
4. 游戏是否处于正常运行状态

## 自定义配置

可以在 `config/aimod-common.toml` 文件中调整相关参数（如果需要的话）。
